## Overview

poidplume66 empowers you to create, organize, and maintain comprehensive documentation for your projects. You get a centralized space to collaborate with your team, version control your docs, and deliver polished, searchable content. Whether you manage APIs, user guides, or internal wikis, poidplume66 streamlines the process.

<Callout kind="info">
  poidplume66 supports Markdown, MDX, and integrates with Git for seamless workflows.
</Callout>

## Getting Started

Begin with these simple steps to launch your documentation.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://poidplume66.com` with your email.
  </Step>
  <Step title="New Project" icon="plus">
    Click "New Project" and name it, like "My API Docs".
  </Step>
  <Step title="Add First Doc" icon="file-plus">
    Create `introduction.mdx` and start writing.

````mdx
---
title: My First Doc
description: Hello world example.
---

## Hello poidplume66

Welcome to your documentation!
````

  </Step>
  <Step title="Publish" icon="globe">
    Hit "Publish" to generate your live site at `https://docs.poidplume66.com/my-project`.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
```javascript
// Fetch your docs via API
const response = await fetch('https://api.example.com/docs/my-project', {
  headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
});
const docs = await response.json();
console.log(docs);
```
```python
# Fetch docs using requests
import requests
response = requests.get(
    'https://api.example.com/docs/my-project',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
)
docs = response.json()
print(docs)
```
</CodeGroup>
