## How to Upgrade

Follow these steps to update poidplume66 seamlessly.

<Steps>
  <Step title="Backup" icon="database">
    Export your current documentation space.
  </Step>
  <Step title="Update" icon="download">
    
````bash
npm update poidplume66@latest
````

    Or via Docker:
    
````dockerfile
image: poidplume66:latest
````
  </Step>
  <Step title="Restart" icon="refresh-cw">
    Restart your instance and verify changes.
  </Step>
</Steps>

<Update label="2024-08-10" description="v1.3.0" tags={["bugfix", "breaking"]}>

<Callout kind="alert">
  This release includes breaking changes to the export API. Review migration steps below.
</Callout>

## Improvements

- PDF and HTML export with custom templates

## Bug Fixes

- Fixed authentication token expiration handling
- Improved error logging for failed builds

## Breaking Changes

- Deprecated `legacyExport` endpoint; migrate to `v2Export`

<Expandable title="Migration Guide" default-open="true">
  
````javascript
// Old
fetch('https://api.example.com/legacyExport')

// New
fetch('https://api.example.com/v2Export', {
  body: JSON.stringify({ format: 'pdf' })
})
````
</Expandable>

</Update>
