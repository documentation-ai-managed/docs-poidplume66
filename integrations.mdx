## Overview

poidplume66 supports seamless integrations with popular third-party services, webhooks for real-time notifications, full API access for custom builds, and flexible export/import options. You can automate workflows, sync content across platforms, and extend functionality without leaving your documentation space.

<Callout kind="info">
  All integrations require a valid API key from your poidplume66 dashboard. Generate one at `https://dashboard.example.com/settings/api`.
</Callout>

## Webhook Setup

Set up webhooks to get instant notifications for events like document updates or user actions.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to `https://dashboard.example.com/settings/webhooks` and click "New Webhook".
  </Step>
  <Step title="Configure URL" icon="link">
    Enter your endpoint URL, e.g., `https://your-webhook-url.com/poidplume66`.
  </Step>
  <Step title="Select Events" icon="bell">
    Choose events like `document.updated` or `space.member_added`.
  </Step>
  <Step title="Test and Save" icon="check-circle">
    Send a test payload and save the webhook.
  </Step>
</Steps>

Verify payloads with these examples:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Example incoming webhook payload
  {
    "event": "document.updated",
    "space_id": "spc_123abc",
    "document_id": "doc_456def",
    "timestamp": 1699123456,
    "user": {
      "id": "usr_789ghi",
      "email": "user@example.com"
    }
  }
  ```
  ```python
  # Process webhook in Python
  import json
  from flask import Flask, request

  app = Flask(__name__)

  @app.route('/webhook/poidplume66', methods=['POST'])
  def webhook():
      data = request.json
      if data['event'] == 'document.updated':
          print(f"Document {data['document_id']} updated")
      return '', 200
  ```
</CodeGroup>

## API Access for Custom Integrations

Build custom integrations using poidplume66's REST API at `https://api.example.com/v1`.

<Request tabs="JavaScript,cURL" show-lines="true">
  ```javascript
  const response = await fetch('https://api.example.com/v1/documents', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title: 'New Document',
      content: '# Hello from API'
    })
  });
  ```
  ```bash
  curl -X POST https://api.example.com/v1/documents \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "title": "New Document",
      "content": "# Hello from API"
    }'
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "id": "doc_456def",
    "title": "New Document",
    "content": "# Hello from API",
    "space_id": "spc_123abc",
    "created_at": "2023-11-01T10:00:00Z"
  }
  ```
</Response>

<ParamField path="title" param-type="string" required="true">
  Document title, max 255 characters.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token using your API key.
</ParamField>

## Export and Import Options

Export your documentation as Markdown, PDF, or HTML. Import from GitHub or Google Docs.

<Expandable title="Advanced Export Settings" default-open="false">

Use the API for bulk exports:

```bash
curl -X GET "https://api.example.com/v1/export/space/spc_123abc?format=markdown" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  --output docs.zip
```

</Expandable>

<Callout kind="tip">
  For large imports, use the dashboard uploader at `https://dashboard.example.com/import` to avoid API rate limits.
</Callout>
